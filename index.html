let, var, const

+

🎓 Let's Learn About JavaScript Variables!

🌟 var - The Old Way

Think of 'var' like a magic box that can hold anything and can be changed anytime!


var myAge = 10;     // We put 10 in the box
myAge = 11;         // We can change it to 11
var myAge = 12;     // We can even make a new box with the same name!

🔒 const - The Never-Change Box

const is like a special treasure box - once you put something in, it stays forever!


const myName = "Sara";    // Put "Sara" in a locked box
// myName = "John";       // Oops! Can't change it - it's locked!
const favColors = ["blue", "red"];  // We can put a list in the box
favColors.push("green");   // We can change what's inside the list
// favColors = ["yellow"]; // But we can't replace the whole list!

🎈 let - The Modern Way

let is like a friendly box - you can change what's inside, but you can't make another box with the same name!


let myPets = 2;     // Start with 2 pets
myPets = 3;         // Got a new pet! Now we have 3
// let myPets = 4;  // Can't make a new box with same name!

🤔 When to Use What?

  • 🔒 Use const when you don't want the value to change (like your name)
  • 🎈 Use let when you might need to change the value later (like your age)
  • ⚠️ Try not to use var - it's old and can cause confusion!

Promises

+

🤝 Promises - JavaScript's Way of Making Promises!

🎭 What is a Promise?

A Promise is like ordering a pizza! When you order, the restaurant PROMISES to either:

  • ✅ Bring you your pizza (Promise fulfilled - resolve)
  • ❌ Tell you they can't make it (Promise rejected - reject)

// Making a simple promise (like ordering a pizza)
let pizzaOrder = new Promise((resolve, reject) => {
    let pizzaIsReady = true;  // Is the pizza ready?
    
    if (pizzaIsReady) {
        resolve("🍕 Here's your delicious pizza!");
    } else {
        reject("😢 Sorry, we're out of cheese!");
    }
});

🎯 Using Promises

🎪 then, catch, and finally

pizzaOrder
    .then((message) => {
        console.log(message);  // "🍕 Here's your delicious pizza!"
    })
    .catch((error) => {
        console.log(error);    // "😢 Sorry, we're out of cheese!"
    })
    .finally(() => {
        console.log("👋 Thank you for visiting our pizza shop!");
    });

🎭 Promise States

A Promise can be in one of three states:

  • ⏳ Pending - Still waiting (like when the pizza is cooking)
  • ✅ Fulfilled - Success! (pizza delivered)
  • ❌ Rejected - Something went wrong (no more cheese)

💫 Promise Magic Tricks


// Wait for multiple promises (like ordering many pizzas!)
Promise.all([
    playLuckyDraw(),
    loadPlayerProfile("123")
])
    .then(([luckResult, player]) => {
        console.log("🎉 Everything is ready!");
    })
    .catch(error => {
        console.log("😅 Something went wrong!");

// Race promises (first one wins!)
Promise.race([
    playLuckyDraw(),
    loadPlayerProfile("123")
])
    .then(winner => {
        console.log("🏃‍♂️ This one finished first:", winner);
    });

  • ⚡ Promises are great for loading data or waiting for something
  • 🎮 Use Promise.all() when you need multiple things to finish
  • 🏃‍♂️ Use Promise.race() when you want the fastest result

Asynchronous JavaScript

+

⚡ Asynchronous JavaScript - Doing Many Things at Once!

🎭 What is Asynchronous JavaScript?

Imagine you're at a restaurant:

  • 🍽️ You order food (start a task)
  • 🎮 While waiting, you can play on your phone (do other things)
  • 🔔 When food is ready, the waiter calls you (task complete!)

That's exactly how asynchronous JavaScript works - it can do many things without waiting!

🎯 Ways to Write Asynchronous Code

1️⃣ Callbacks - The Old Way

// Using callbacks (old style)
console.log("👨‍🍳 Starting to cook pasta...");
setTimeout(function() {
    console.log("🍝 Pasta is ready!");
    setTimeout(function() {
        console.log("🧀 Adding cheese!");
        setTimeout(function() {
            console.log("🍽️ Dinner is served!");
        }, 1000);
    }, 1000);
}, 2000);
2️⃣ Promises - The Better Way

// Same thing with promises
function cookPasta() {
    return new Promise(resolve => {
        setTimeout(() => {
            resolve("🍝 Pasta is ready!");
        }, 2000);
    });
}

function addCheese() {
    return new Promise(resolve => {
        setTimeout(() => {
            resolve("🧀 Added cheese!");
        }, 1000);
    });
}

console.log("👨‍🍳 Starting to cook pasta...");
cookPasta()
    .then(message => {
        console.log(message);
        return addCheese();
    })
    .then(message => {
        console.log(message);
        console.log("🍽️ Dinner is served!");
    });
3️⃣ Async/Await - The Modern Way

// Same thing with async/await (super clean!)
async function makeDinner() {
    console.log("👨‍🍳 Starting to cook pasta...");
    const pasta = await cookPasta();
    console.log(pasta);
    const cheese = await addCheese();
    console.log(cheese);
    console.log("🍽️ Dinner is served!");
}
